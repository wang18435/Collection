IDEA安装
从安装上来看，对硬件的要求不是很高。
可在实际开发中，IDEA执行时会有大量的缓存，索引文件，如果使用eclipse卡顿，使用IDEA也是必然卡顿的
安迪比尔定律
安迪->安迪格鲁夫 英特尔CEO   硬件提高的性能，很快就会被软件消耗掉

安装目录和设置目录结构
bin下   启动文件包扩虚拟机的配置信息 以及IDEA的属性
    在之下，有idea64.exe.vmoptions该文件
        -Xms 初始内存数   设置变大，可提高启动速度
        -Xms 最大内存数   1500  降低垃圾回收的频率
        -XX ReservedCodeCacheSize 保留的代码缓存大小   可以减少无用代码回收的频率

help   帮助文档
jre64  提供的java运行环境
lib   IDEA依赖的相关类库
license   相关的插件许可信息

模块(Module)
在IDEA中只有Project(工程)和Module（模块）的概念。
一个Project可以打开一个窗口，多项目管理就得打开多个窗口。
为什么要这么设置？
    目前的主流的大型项目都是分布式部署的，结构都是项目下多个moudle结构。
    例如：core Module,Web Module,plugin Module,solr Module
    Module的删除
        open Module settings
        移除之后（删除模块，它就不是模块，但它仍然存在硬盘上），之后便可以删除
        生成的字节码文件在out目录下。

常用配置
    在settings中（进入设置界面）
（1）外观和行为
    主题：外观（有三套）
    设置窗体及菜单的大小及字体大小
    设置编辑区域主题
        IDEA设置了两个编辑区主题
        editor->color scheme 
        更多效果：
        http://www.riaway.com     下载以后，导入主题，file->import settings->选中下载的jar文案->一路确认->重启,重启之后
        新主题酒会启用，如果没有启用，可以如下方式选择。或者实在color scheme中导入主题
（2）Edit-General
    设置鼠标滚轮修改字体大小
    设置鼠标悬浮提示（quick doucumentations） 可设置时间
    Auto import
    设置行号和方法间的分隔符
    忽略大小写提示 （case sensitive completion）
    设置取消单行显示tabs的操作   （多行显示）
（3）Editor-Font设置默认的字体，字体大小，字体行间距
（4）Editor-Color scheme 修改当前主题的字体，字体大小，字体间行间距
    修改代码中注释字体的颜色         Language Defaults ->Line comment  单行注释  block（多行注释）
（5）超多指定的import个数，改为*
（6）修改类头的文档注释信息
    File and Code Templates——>dile Header 
    /**
    @Author xxx
    @create ${year}-${}-${Day}-${Time}
    */
（7）设置项目文件编码
    设置文件编码
        Editor->File Encodings
        Reload 本身文件的编码形式不变，仅改变显示
        convert 直接转化 
（8）设置自动编译 Build,Excution,Deployment
（9）设置为省电模式 file ->Power save mode 关闭代码检查和代码提示的功能
（10）设置代码水平过垂直显示
